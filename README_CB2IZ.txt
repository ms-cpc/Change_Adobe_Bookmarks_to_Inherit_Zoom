:: DESCRIPTION: Simple batch to automate drag and drop application for changing bookmarks to inherit zoon in a PDF. Useful when in court with low resolution monitors so that bookmarks do not reset to fit page.
:: ######################################################

HOW TO USE:
1) When bookmarking your PDF, ensure you click the "fit to page button" and do not zoom in or out when creating bookmarks. This will make all the bookmarks you make "fit to page"

2) Run CB2IZ_MS.bat

3) Drag your PDF into the cmd prompt, click that window and press enter

4) type new file name (no .pdf extension, this is added automatically)

5) PDF is created in the CB2IZ folder (too lazy to place it back in original folder, sorry). It will open and you can re-save it to the appropriate folder, or you can move it.

6) Clean up your CB2IZ folder of old reports.
